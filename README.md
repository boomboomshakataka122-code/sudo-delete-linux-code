# sudo-delete-linux

A playful Linux script that simulates deleting your system in a dramatic way—completely harmless.  

Run it, watch the output, and feel like a root-level hacker without risking anything.  

## Usage

You could use it for nothing. Really. It wont destroy your system, it wont do anything. You could prank a new linux user!

## How to install it and use it
The commands are :
git clone https://github.com/boomboomshakataka122-code/sudo-delete-linux-code.git
cd sudo-delete-linux
sudo chmod +x delete
sudo mv delete /usr/local/bin (optional, if you want to remove ./ from ./delete)
sudo delete linux
